Services

Website Development

Utilizing agile web development methods, first I communicate. Getting a good understanding of the clients vision is paramount for speedy development.

Website Design

Next, I iterate a design depending on the project. Figma and Mockup are my go to programs for creating fast mockups of the designs. These mockups also help massively improve the pace on specific changes that are required.

Website Deployment

Finally, once the client is satisfied with how it is created visually, I take the final step and create the website. HTML, CSS, and Bootstrap are my tools of choice, but for slightly more complex projects, I add on Java for additional functionality. Then it's done!

Game Design

First, I identify what the core loop of the game is, and put it into a overall design document. Most of the time, these game design documents end up looking like a road plan.

Game Prototyping

Then, once the plan has been made, we make working prototypes. This step usually requires rewrites and retooling of mechanic. Unlike Web development, you can only vizualize game design concepts to a certain point.

QA

After the prototype has a working gameplay loop, than it is time to polish mechanics and fix errors in a testing driven design philosiphy. This takes by far the longest, but is the most fufilling!
